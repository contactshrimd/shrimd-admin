# Tech Stack (Admin Portal)

## Framework
- React (web)
- TypeScript (strict mode)

## Hosting
- Vercel

## UI
- Tailwind CSS
- Component library (optional: ShadCN / Material UI)

## State Management
- React Query (server state)
- Context or Zustand (UI state)

## Auth
- Firebase Auth (custom claims for roles)

## API
- Axios or Fetch wrapper

---

## Constraints

- No direct database access
- All data via backend APIs
- No PHI stored locally
